# NovaPulse — AI-Powered Personal Finance Platform

A premium, dark-themed landing page for NovaPulse, an AI-powered personal finance platform. Built with Next.js 14, TypeScript, and Tailwind CSS.

![NovaPulse Preview](preview.png)

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Icons:** Custom SVG icons + Lucide React
- **Fonts:** Inter (Google Fonts)
- **Animations:** Custom CSS animations + Intersection Observer API
